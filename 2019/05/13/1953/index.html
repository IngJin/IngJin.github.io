190516 TIL

  • JDBC 프로젝트, 어렵다.

  • 외부 API 등록하는 방법1
    1.등록하고자 하는 API(라이브러리)를 복사한다.
    2.C:\Program Files\Java\jre1.8.0_202\lib\ext 붙여넣기
    3.Eclipse 재시작(File MenuBar > Restart)
    4.모든 프로젝트에서 해당 API를 활용할 수 있다.
    5.Export 할 때 별도로 해당 라이브러리를 내보내기 해야한다.

  • 외부 API 등록하는 방법2(권장)
    1.해당 프로젝트 > 우클릭 > Build Path > Configure Build Path
    2.상단 탭 : Libraries > Add External JARs… > 해당 라이브러리를 선택 > Apply and Close
    3.해당 프로젝트에서만 API를 활용할 수 있다.
    4.Export시 해당 라이브러리도 같이 내보내기 된다.

  • 방법1과 방법2를 혼용할 시 오류가 발생할 수 있다
    : 둘 중 하나를 삭제
